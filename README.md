# Numpy taak - Stijn Van Vlaenderen 
## 1 setup
### 1. 1 docker image  
Docker image gebruikt om deze code te runnen is te vinden op:  
```console
docker pull stijnvanvl/taaknumpy:latest
```
```console
docker run -it stijnvanvl/taaknumpy:latest
```
